# Архив калькулятора баннеров

Дата архивации: 2025-11-23

## Описание
Калькулятор печати на баннере с визуальным редактором макета.

## Основные возможности
- Расчёт площади баннера в м² по размерам в сантиметрах
- Опция добавления люверсов 10 мм по периметру
- Визуальный редактор макета с настройками:
  - Текст
  - Шрифт (Arial, Montserrat, Courier New, Georgia, Verdana)
  - Размер шрифта (12-120px)
  - Горизонтальное выравнивание (left, center, right)
  - Вертикальное выравнивание (top, center, bottom)
  - Цвет фона
  - Цвет текста
- Превью макета с соотношением сторон 16:9
- Автоматическое заполнение формы контактов данными калькулятора
- Минимальная площадь для печати: 2 м²

## Интеграция
Компонент был интегрирован в секцию CalculatorSection как один из трёх калькуляторов (баннеры, инфостенды, вывески).

## Технические детали
- React компонент с TypeScript
- Использует shadcn/ui компоненты (Button, Card, Input, Select, Checkbox)
- State management через useState и useRef
- Расчёт площади: (ширина × высота) / 10000
- Изображение люверса: https://cdn.poehali.dev/projects/820f24d3-2a0c-446f-996e-d0f46f8895f8/files/58bc4c75-c28a-4749-8852-f6a438d451f8.jpg
